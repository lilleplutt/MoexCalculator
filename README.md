## **MoexCalculator**

MoexCalculator — мини‑приложение для конвертации валют по данным Московской биржи

## **Описание приложения**

- Мини‑конвертер валют, который отображает актуальные курсы и пересчитывает суммы между валютами.
- Данные загружаются с MOEX, курсы обновляются и применяются автоматически.
- Интерфейс реализован на SwiftUI с минималистичной логикой взаимодействия.

## **Технические особенности**

- Язык реализации — Swift;
- UI-фреймворк — SwiftUI;
- Реактивные потоки — Combine;
- Загрузка данных — MOEX ISS API.
